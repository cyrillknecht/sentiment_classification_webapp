About

This webapp was made with Flask and deployed on Google Cloud using Google Kubernetes Engine (GKE). It classifies the sentiment of input sentences as POSITIVE or NEGATIVE using the distilbert-base-uncased-finetuned-sst-2-english model provided by Huggingface.
